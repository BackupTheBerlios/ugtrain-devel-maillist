Our goal is to integrate scanmem into ugtrain so that also for
discovery no root priviledges are required anymore. With that
the whole cheating process from discovery to adaption can be
done as a regular user with ugtrain as a commander application.

It turned out that a lot of system specific enhancements are
required for that. So we rework and enhance that part. We
introduce a function to run two commands as a process group in
the background as scanmem must be the parent of the game in
order to bypass OS security. The downside is that we have to
care for orphans.

In case scanmem is run externally, then we don't want to
interfere as both use ptrace(). The idea is now not to attach
to game memory if all configs are disabled.
